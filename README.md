手軽にアンケートを作成できます。
データ集、権威性を持たせる際のデータ使用

# Requirement

* php 7.4
* laravel 7.30
* apache 2.4.48
* mysql Ver 15.1 Distrib 5.5.68-MariaDB

# Note

投票はcookieで制限しているので、投票結果はアバウトです。今後ログイン機能で投票を制御する予定です。
気軽にできる様にログイン機能を持たせずに、投票精度を上げたいです。
